# Release Notes

This page records platform and documentation changes that affect readers, contributors, and maintainers.

## Current Release

### Version 2.1.0 - January 20, 2026

#### Highlights
- Introduced multi-sidebar navigation for Guides, Writing, Cloud, and DevOps.
- Improved search experience through Algolia DocSearch integration.
- Standardized heading conventions and metadata formatting across docs.

#### Fixes
- Corrected sidebar bindings and frontmatter rendering behavior.
- Resolved inconsistent heading capitalization in key pages.

## Previous Releases

### Version 2.0.0 - January 11, 2026

#### Highlights
- Added a complete interactive components library.
- Published a full MDX/Markdown reference and quick reference index.

#### Fixes
- Resolved component rendering and MDX parsing issues on smaller screens.

### Version 1.5.0 - January 8, 2026

#### Highlights
- Added dedicated DevOps and Cloud documentation sections.
- Expanded practical examples for CI/CD, containers, and IaC topics.

#### Fixes
- Corrected broken cross-links and syntax highlighting inconsistencies.

### Version 1.0.0 - December 26, 2025

#### Highlights
- First stable release of the TechDOCS portal.
- Published foundational guides for users, admins, integrations, and API references.

## Upgrade Notes

When moving between major feature releases:

1. Clear browser cache to load updated static assets.
2. Revalidate custom sidebar or theme overrides.
3. Run a link and build check before publishing internal mirrors.

## Roadmap Direction

Planned enhancements include:

- stronger API reference tooling,
- richer media support for tutorials,
- and contributor workflow improvements.
