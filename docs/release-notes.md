# Release Notes

Welcome to the Release Notes section of Tech Docs!

## Overview

This section contains version history, new features, bug fixes, and improvements for our products and documentation.

---

## Version 2.1.0 - January 20, 2026

### New Features
- **Multi-Sidebar Architecture**: Implemented separate sidebars for different documentation sections (Guides, Cloud, DevOps, Writing)
- **Algolia Search Integration**: Added Algolia DocSearch for enhanced search functionality
- **Title Case Standardization**: All headers across documentation now follow proper Title Case conventions
- **Emoji Removal**: Cleaned up all emoji characters for professional presentation

### Improvements
- Enhanced navigation with context-aware sidebars
- Improved documentation structure and organization
- Better frontmatter formatting across all MD files
- Optimized sidebar configuration for better UX

### Bug Fixes
- Fixed frontmatter rendering issues causing metadata to display as page content
- Corrected sidebar assignments for cloud and DevOps documentation
- Resolved Title Case inconsistencies in headers

## Migration & Upgrade Notes

### Upgrading to v2.1.0
- Clear browser cache to see updated sidebar structure
- Review custom sidebar configurations if any
- Update any custom CSS that may conflict with new structure

### Upgrading to v2.0.0
- MDX files may need component imports updated
- Review interactive component usage for breaking changes
- Test all custom components after upgrade
